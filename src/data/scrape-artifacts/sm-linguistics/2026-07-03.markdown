# Master of Science in Linguistics

[Department of Linguistics and Philosophy](https://catalog.mit.edu/schools/humanities-arts-social-sciences/linguistics-philosophy/#linguistics-ms)

## Program Requirements

|     |     |     |
| --- | --- | --- |
| [24.900](https://catalog.mit.edu/search/?P=24.900 "24.900") | Introduction to Linguistics 1 | 12 |

| [24.931](https://catalog.mit.edu/search/?P=24.931 "24.931") | Language and Its Structure I: Phonology 1 |  |
| [24.932](https://catalog.mit.edu/search/?P=24.932 "24.932") | Language and Its Structure II: Syntax 1 |  |

| [24.921](https://catalog.mit.edu/search/?P=24.921 "24.921") | Independent Study: Linguistics (Focused on the student's language) 2 | 24 |

| Total Units | 120-144 |

|     |     |
