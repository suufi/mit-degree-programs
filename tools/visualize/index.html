Degree requirement visualizer

Generate a standalone HTML tree for a program:

npm run visualize -- --program 6-7

Output: tools/visualize/out/<program>.html — open that file in a browser.

Markdown fallback:

npm run visualize:md -- --program 6-7

Output: docs/degrees/<program>.md

Review bundle for all programs:

npm run review:bundle

Output: docs/review/index.html, docs/review/manifest.csv, and one HTML file per program under docs/review/programs/.
